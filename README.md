# 🧮 MathSphere

> **Your Complete Mathematics Toolkit in One App**

MathSphere is a modern Flutter application that combines multiple mathematical tools into a single, easy-to-use platform. Instead of switching between different apps for calculations, graphing, conversions, and formulas, MathSphere provides everything in one place.

Perfect for students, teachers, engineers, and anyone who works with mathematics.

---

## ✨ Features

### 🧮 Calculators

* Simple Calculator
* Scientific Calculator
* Financial Calculator

  * Simple Interest
  * Compound Interest
  * EMI Calculator
  * SIP Calculator
  * Loan Calculator

### 📐 Algebra & Calculus

* Equation Solver
* Matrix Calculator

  * Matrix Addition
  * Matrix Subtraction
  * Matrix Multiplication
  * Dynamic Matrix Size
* Calculus Tools
* Complex Number Calculator

### 📊 Graph & Statistics

* Graph Plotter
* Statistics Calculator
* Probability Calculator

### 📏 Geometry & Trigonometry

* 2D Geometry Calculator
* 3D Geometry Calculator
* Mensuration Calculator
* Trigonometry Calculator

### 🔄 Utilities

* Unit Converter
* Formula Library
* Math Notes

## 📥 Installation

Download **Math Sphere** from the Google Play Store.

Or clone the repository:

```bash
git clone https://github.com/your-username/mathsphere.git
```

Install dependencies:

```bash
flutter pub get
```

Run the project:

```bash
flutter run
```
